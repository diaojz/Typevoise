# 麦克风选择功能使用指南

## 功能概述

Typevoise 现在支持选择特定的麦克风设备进行语音识别。这对于以下场景特别有用：

- 使用外接麦克风（如耳机麦克风）
- 电脑内置麦克风距离太远
- 需要在多个麦克风设备之间切换

## 如何使用

### 1. 打开设置

在 Typevoise 主窗口中，点击左侧边栏的"设置"。

### 2. 找到麦克风设备部分

在设置页面中，向下滚动找到"麦克风设备"卡片。

### 3. 选择麦克风

点击"选择麦克风..."按钮，会弹出麦克风选择器窗口。

### 4. 选择合适的设备

在弹出的窗口中：
- 默认选项是"自动检测"，使用系统默认麦克风
- 列表中会显示所有可用的输入设备
- 推荐的设备会有"推荐"标签
- 选中的设备会显示蓝色高亮

### 5. 确认选择

选择好麦克风后，点击"确定"按钮保存设置。

## 常见问题

### Q: 为什么识别不到声音？

可能的原因：
1. **麦克风距离太远**：建议距离 10-30cm
2. **选错了设备**：如果使用耳机，确保选择了耳机麦克风而不是电脑内置麦克风
3. **权限未授予**：检查系统设置 → 隐私与安全性 → 麦克风

### Q: 如何知道选择了哪个麦克风？

在设置页面的"麦克风设备"部分，会显示"当前麦克风"的名称。

### Q: 设备列表为空怎么办？

点击"刷新设备列表"按钮重新扫描。

### Q: 更改麦克风后需要重启应用吗？

不需要。设置会立即生效，下次录音时就会使用新选择的麦克风。

## 技术实现

### 核心组件

1. **MicrophoneManager**：管理麦克风设备的枚举和选择
   - 使用 CoreAudio API 枚举所有输入设备
   - 自动识别系统默认设备
   - 持久化用户选择

2. **SpeechRecognizer**：在录音前设置选定的麦克风
   - 读取用户选择的设备 ID
   - 通过 AudioObjectSetPropertyData 设置为当前输入设备

3. **MicrophonePickerView**：友好的麦克风选择界面
   - 类似 Typeless 的设计
   - 显示设备名称和推荐标签
   - 实时音量指示器（占位）

### 数据持久化

用户选择的麦克风 ID 保存在 UserDefaults 中：
```swift
SettingsManager.shared.selectedMicrophoneID
```
