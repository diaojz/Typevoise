# Typevoise：用 Claude API 复刻 Typeless

---

## 幻灯片 1：标题页

**Typevoise**

用自己的 Claude API 实现语音转文字 + AI 润色

副标题：2 小时从零开发一个 macOS 语音输入工具

---

## 幻灯片 2：为什么要做这个项目？

### 问题：Typeless 太贵了

- 💰 12 美元/月
- 📅 必须按年付费：144 美元/年
- 🔒 功能固定，无法自定义

### 我们的解决方案

- ✅ 使用自己的 Claude API Key（日常已在用）
- ✅ 成本几乎为零（按实际用量计费）
- ✅ 完全可控，可自定义扩展

**核心思路**：既然有 API，为什么不自己做一个？

---

## 幻灯片 3：产品定位

### 这是什么？

一个 macOS 菜单栏应用，通过全局快捷键实现：

1. 语音转文字（macOS 系统识别，免费）
2. AI 智能润色（Claude API）
3. 自动粘贴到光标位置

### 适用场景

- 写邮件、微信聊天
- 编辑文档、做笔记
- 任何需要文字输入的地方

---

## 幻灯片 4：用户体验流程

### 4 步完成语音输入

```
1. 在任意 App（微信/邮件/文档）
   ↓
2. 按快捷键（⌘⇧V）
   ↓
3. 说话，再按快捷键停止
   ↓
4. AI 润色后的文字自动粘贴
```

### 效果示例

**说**："嗯那个今天呢我想跟你说一下项目进度"

**得到**："今天想和你汇报一下项目进度。"

## 幻灯片 5：核心功能（MVP）

### 1. 按快捷键录音
- 按一次开始，再按一次停止
- 默认快捷键：⌘⇧V

### 2. 语音转文字
- 使用 macOS 系统语音识别
- 完全免费，无需额外成本

### 3. AI 润色
- 调用 Claude API
- 口语转书面语，自动优化表达

### 4. 自动粘贴
- 文字自动出现在当前光标位置
- 支持任何应用

## 幻灯片 9：技术架构

### 简洁的技术栈

```
快捷键 → 语音识别 → Claude API → 自动粘贴
```

### 核心模块

1. **HotkeyManager**：监听全局快捷键
2. **SpeechRecognizer**：调用系统语音识别
3. **ClaudeService**：调用 Claude API 润色
4. **TextInserter**：模拟键盘粘贴

**技术选型**：
- Swift + SwiftUI
- macOS 系统 API（Speech Framework、CoreGraphics）
- Claude API
