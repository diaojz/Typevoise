# Typevoise 产品需求文档

> 复刻 Typeless App：用自己的 Claude API 实现语音转文字 + AI 润色

---

## 为什么做这个？

**Typeless 太贵了**：12 美元/月，还必须按年付费（144 美元/年）

**我们的优势**：
- 已经有 Claude API Key（日常在用）
- 使用成本几乎为零（按实际用量计费）
- 可以自定义和扩展功能

**目标**：2 小时内完成核心功能

---

## 核心功能（MVP）

### 1. 按快捷键录音
```
按 ⌘⇧V → 开始录音 → 再按一次 → 停止
```

### 2. 语音转文字
使用 macOS 系统语音识别（免费）

### 3. AI 润色
调用 Claude API 把口语转成书面语

### 4. 自动粘贴
文字自动出现在当前光标位置

---

## 用户流程

```
在任意 App（微信/邮件/文档）
        ↓
    按快捷键
        ↓
      说话
        ↓
    再按快捷键
        ↓
  AI 润色后的文字自动粘贴
```

**示例**：
```
说："嗯那个今天呢我想跟你说一下项目进度"
得到："今天想和你汇报一下项目进度。"
```

## 技术架构

```
快捷键 → 语音识别 → Claude API → 自动粘贴
```

### 核心模块

1. **HotkeyManager**：监听全局快捷键
2. **SpeechRecognizer**：调用系统语音识别
3. **ClaudeService**：调用 Claude API 润色
4. **TextInserter**：模拟键盘粘贴
